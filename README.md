# Pro PDF Mate

Professional PDF tools platform with production-ready Express backend and responsive UI.

## UI & branding improvements
- Navbar with logo (`/public/images/logo.png`)
- Responsive menu (desktop + hamburger mobile)
- Consistent tool page template and spacing
- Reusable upload component with:
  - drag/drop
  - file preview
  - progress bar
  - processing state
  - download result area

## Main pages
- Home: `/`
- Tools catalog: `/all-tools`
- Core SEO tools: `/merge-pdf`, `/split-pdf`, `/compress-pdf`, `/pdf-to-word`, `/jpg-to-pdf`
- Categories: `/convert-pdf`, `/edit-pdf`, `/optimize-pdf`, `/security-pdf`

## Tool coverage in UI
- Merge, Split, Compress, Rotate, Organize, Delete Pages, Extract Pages
- JPG/PNG/Word/Excel/PowerPoint/HTML to PDF
- PDF to JPG/PNG/Word/Excel/PowerPoint/Text
- Protect, Unlock, Remove restrictions, Sign
- Repair, Reduce size, Extract Images, OCR, Flatten

## Run
```bash
npm install
npm start
```
